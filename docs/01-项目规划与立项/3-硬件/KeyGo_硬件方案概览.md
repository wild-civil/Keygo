## 0. 与立项稿的主要变更

| 项 | 立项稿（旧） | 本概览（新） | 原因 |
|---|---|---|---|
| 充电 IC | TP4056（线性，无温控） | **CN3165**（线性 + VIN-DPM，带 TEMP/NTC 温控） | 车载 >45°C 硬停充保命；5V 小板 Vmp≈钳位点 |
| 电池 | 18650 | **14500 或 聚合物 Li-Po**（宽温 + 保护板） | 看钥匙壳空间取舍；均 3.6~4.2V Li-ion |
| LDO | ME6211 | **XC6206**（或 MCP1700，更低 Iq） | 深睡静态电流不被 LDO 吃掉 |
| 太阳能板 | 5V/1W | **5V/100mA（或 200mA）小板** | 功耗低，小板够；避免大板弱光浪费 |
| 作动器 | TMUX1112（4 通道） | **3 路 MOS（或保留 TMUX 选项，见 §3）** | 按钮是信号级，MOS 够用；但 MOS 有体二极管陷阱 |
| 钥匙供电 | TMUX 第 4 路给 fob VCC | **LDO 3.3V → PMOS 高侧开关 → fob VCC** | 明确由控制板供 fob 电，断 fob 电池省维护 |
| 温控/遥测 | 无 | **CN3165 TEMP+NTC 硬停充 + CH582M TSENSE→APP `t`** | 车载安全 + 温度显示 |

> ⚠️ 化学体系锁定 **Li-ion（3.6~4.2V）**，**不用 LFP**（LFP=3.2/3.65V 需换 TP5000+改稳压）。14500/聚合物皆是 Li-ion，电气无差异。

## 1. 系统框图（最新）

```
[5V 太阳能板] --(TVS 钳位)--> VIN
                            [CN3165]  -- BAT --> [电池 保护板/14500或Li-Po]  -- BUS(3.0~4.2V)
                               |  TEMP <-- NTC(贴电池)                          |
                               | CHRG/DONE(悬空)                                |
                          BUS --+--> [XC6206 LDO] --> 3.3V 主电源
                                |                 |
                                |     ┌───────────┼──────────────┐
                                |     │           │              │
                                |  [CH582M]   [TMUX1112 或       [PMOS 高侧开关]
                                |  VDD       3×MOS] 按键模拟      源极=3.3V
                                |  (TSENSE)    │                 栅极<-GPIO(经电阻)
                                |              │                 漏极--> fob VCC
                                |              │                        │
                                |              └── 跨接 fob 按键触点      │
                                |                                        ▼
                                |                                    [原车钥匙 PCB]
                                |                                    (断原电池,由板供电)
                                └── 电池电压 ADC 分压(高阻/PMOS门控)
```

- 拓扑：**并联母线自动切换**（有太阳灌电、无太阳用电池，无切换开关）。
- 负载永远在电池母线上，3.3V 由 LDO 出，不受面板波动影响。

## 2. 电源方案

- **太阳能 → 充电**：5V/100mA（或 200mA）板 → CN3165（VIN-DPM 把输入钳 ~4.4V≈面板 MPP）→ 电池母线。
  - RISET：`100mA→12kΩ`；`200mA→6.2kΩ`（1% 精度；只设电流天花板，太阳能下被 VIN-DPM 自动限流）。
  - TVS：VIN↔GND 串 **BZT52C5V6**（5.6V），钳 5V 板 Voc≈6V，防浪涌。
  - CIN/CBAT：10µF X7R 16/25V，紧贴引脚，单点接地。
- **电池**：14500 或聚合物 Li-Po（宽温放电款 + 保护板，过充/过放/短路）。阴凉放置（手套箱/座椅下）。
- **LDO**：XC6206P332MR（Iq≈1µA）→ 3.3V → CH582M / TMUX(MOS) / PMOS 控制。MCP1700 压差更小可作备选。
- **温控**：CN3165 TEMP 脚接 10k NTC(β3950) 贴电池 → **>45°C 自动暂停充电**（芯片硬行为，不依赖 MCU）。
- **电池电压 ADC**：1M+1M 高阻分压或 PMOS 门控 → CH582M ADC（防常泄，监测电量）。

## 3. 钥匙供电与按键模拟（核心改动）

### 3.1 钥匙 fob 由控制板供电（PMOS 高侧开关）

- 断开原车钥匙纽扣电池，由控制板供 fob VCC：
  - `LDO 3.3V → PMOS(如 AO3401, SOT-23) 源极=3.3V → 漏极→ fob VCC`
  - PMOS 栅极经 10k 上拉到 3.3V（默认关断 Vgs=0），CH582M GPIO 经 100Ω 拉低栅极 → Vgs=-3.3V 导通。
- **上电时序**（固件）：供电=1 → 延时 ~200ms（等 fob MCU 初始化）→ 触发按键 → 延时 ~150ms → 释放 → 延时 ~500ms（等 RF 发射）→ 供电=0（断电省电）。
- ⚠ fob 原 CR2032=3V，现供 3.3V 多数遥控 IC 可接受（工作范围 2~3.6V）；若严格 3V，可在 PMOS 后串肖特基降到 ~3.0V，或选 3.0V LDO。

### 3.2 按键模拟：3 路（解锁/锁车/后备箱或骑行）

| 方案 | 元件 | 优点 | 风险 |
|---|---|---|---|
| **A. 3× 低侧 N-MOSFET**（2N7002/AO3400） | 3 颗 SOT-23 | 便宜、简单 | ⚠ **体二极管陷阱**（见下） |
| **B. TMUX1112**（4 通道 SPST） | 1 颗 TSSOP-16 | 双向、无体二极管、漏电流 3pA、先断后合 | 成本略高（~4~5 元） |

**⚠ 体二极管陷阱（方案 A 必须处理）**：
- fob 按钮一端经上拉到 VCC(3.3V)、另一端 GND。单颗 N-MOS 源极接 GND、漏极接按钮信号端时，**关断状态体二极管（D→S）方向与偏压同向 → 信号端被拉到 GND → 误触发按键！**
- 解决：① **背靠背双 MOS**（每路 2 颗，源极相连，体二极管反向抵消）→ 元件翻倍；或 ② 直接选 **TMUX1112**（无此问题，且 1 颗含 4 通道，可把第 4 路省下或做冗余）。

**推荐**：
- 求省料 → 方案 A + 每路背靠背双 MOS（共 6 颗）。
- 求稳/省 PCB 面积 → 方案 B（TMUX1112，1 颗搞定 3 路按键 + 可留 1 路备用）。
- 两种都需 fob 与 KeyGo **共地**（fob GND 引到控制板地）。

## 4. 温度采集与 APP 上报

- **硬件安全**：CN3165 TEMP+NTC 实现 >45°C 硬停充（必做，不依赖 MCU）。
- **软件遥测**：CH582M 内置 TSENSE（ADC 内部通道），固件读温 → FF02 status JSON 加 `"t":<℃>`；APP 复用现有字段显示。
  - 精度：TSENSE ±2~5°C，测结温，**趋势指示够用、绝对数值偏几度**。
  - 可选进阶：NTC 中点再分压一路 → CH582M ADC → `"tb"`（电池表面温度，±0.5~1°C 更准）。
- 无需外部温湿度协议（湿度无传感器）。
