# KeyGo 硬件供电与驱动设计 v0.1（2026-07-18）

> 状态：初稿，待逐项核对数据手册 + 嘉立创实时料号。
> 目标：CH582M 车锁设备，太阳能+锂电池供电，车载高温环境。

## 1. 设计目标与已定假设
- 太阳能板：**5V / 100mA（暂定，亦可 200mA）**，滴胶/PET 小板。
- 充电 IC：**CN3165**（线性 + VIN-DPM 输入钳位 ~4.4V；对 5V 板 Vmp≈4.3V≈钳位点，近似 MPP）。
  - 注：CN3791（Buck）需 Vin≫Vbat，5V 板喂不动，本方案不选。
- 拓扑：**并联母线**（电池常驻 + 太阳能灌电流 + 负载挂母线），无切换开关。
- 作动器：**模拟钥匙按钮 = 信号级短接**，用低侧 N-MOSFET，非电机/电磁阀。
- 环境：车内暴晒 60~80°C，需 NTC 温控 + 高温电池 + X7R 电容。
- EDA：嘉立创 EDA（立创 EDA 标准版）。

## 2. 系统框图
```
[5V 太阳能板] --(TVS 钳位, 可选)--> VIN
                                 [CN3165]  -- BAT --> [电池 保护板/14500或Li-Po]  -- BUS(3.0~4.2V)
                                    |  TEMP <-- NTC(贴电池)                    |
                                    | CHRG/DONE(悬空或10k限流)                 |
                                    |                                         |
                              BUS --+--> [3.3V LDO] --> CH582M VDD
                                    |            (退耦/晶振/天线/RST=PB23/BOOT=PB22)
                                    |
                                    +--> [N-MOSFET x3] 漏极/源极 跨接 钥匙按钮触点
                                         栅极 <-- CH582M GPIO(3.3V) + 下拉
```
- 电池电压 ADC：高阻分压(1M+1M) 或 PMOS 门控 → CH582M ADC（防常泄）。
- 温度遥测：CH582M 内部 TSENSE → FF02 status `"t"`；可选 NTC 中点 tap → `"tb"`。

## 3. 电源部分（CN3165）
参考 CN3165 数据手册典型应用；DFN-8 封装。
- **VIN**：太阳能板 +。建议串 **5.6V 稳压管/TVS**（如 BZT52C5V6 或 SMAJ5.0A）到地，钳制面板开路高压（5V 板 Voc≈6V；若误用 6V 板 Voc 可达 7~8V 超 VIN 上限）。
- **CIN**：10µF X7R 16V/25V 陶瓷（0805/1206），VIN↔GND。
- **BAT**：电池 +。
- **CBAT**：10µF X7R 16V/25V，BAT↔GND。
- **ISET**：经 **RISET** 到 GND，仅设定充电电流（**不能设定 MPPT 钳位点**，钳位点内部固定 ~4.4V）。
  - 公式 `ICH = 1188 / RISET(Ω)`（常数以手头数据手册为准）。
  - 100mA → RISET≈11.9kΩ（取 12kΩ）；200mA → RISET≈5.9kΩ（取 6.2kΩ）。选 1% 精度。
- **TEMP**：接 **10kΩ NTC（β=3950）到地**，NTC 紧贴电池表面。温度越界（典型 >45°C 或 <0°C）自动暂停充电。按数据手册 TEMP 典型电路确认偏置。
- **CHRG / DONE**：开漏状态脚。**建议悬空（省电）**；若指示，限流电阻≥10kΩ + 高亮 LED（电流 ≤0.3mA）。
- **GND**：系统地。

## 4. 电池（14500 圆柱锂离子 vs 聚合物锂电 Li-Po）
- **两者皆 Li-ion 3.6~4.2V 体系**，CN3165（4.2V 终止）+ XC6206（3.3V LDO）完全兼容；与 LFP(3.2/3.65V) 不同，**本方案不用 LFP**。
- 对比：
  | 维度 | 14500 圆柱锂离子 | 聚合物锂电 Li-Po |
  |---|---|---|
  | 电压 | 3.6~4.2V | 3.6~4.2V |
  | 形态 | 标准 AA ⌀14×50mm 刚性 | 可定制薄/异形，铝塑膜柔性 |
  | 容量 | 约 600~1000mAh | 同空间可更高，形状自由 |
  | 宽温/高温工业级 | 易采购（工业级/高温放电 -20~60°C 甚至 70~85°C 款多） | 多消费级；高温工业级需挑厂商，相对少 |
  | 保护板 | 多数不带，选"带保护板"款或外置 DW01+FS8205 | 成品常自带保护板 |
  | 固定 | 电池座/点焊/热缩，需 ⌀14×50 空间 | 双面胶/泡棉贴壳内壁，易塞异形薄空间 |
- **选型建议（取决于钥匙壳内部空间）**：
  - 壳内能塞下 AA（⌀14×50mm）→ 选 **14500**，最简单、宽温工业级好找、可靠。
  - 壳薄/异形/空间紧张 → 选 **超薄聚合物**（按空间定尺寸，如 302030/303040 等），但要挑**宽温款**并确认**带保护板**。
  - 两者电气无差异，按物理空间取舍即可。
- **不要选倍率/动力型**：要低自放、宽温、小容量够用；动力型内阻低但车载无需大电流放电（MOSFET 只信号级）。
- **容量估算**：日均 ~24mAh，100mA 面板 4h 日照≈400mAh/天；14500 或 500~1000mAh 聚合物均可缓冲数十天阴天，不必求大。
- **必须保护**：电芯自带保护板（DW01+FS8205 过充/过放/短路），或外置保护电路。若用普通 DW01 保护板，确认过放阈值适配 Li-ion（~2.5~3.0V），避免过放伤电芯。
- **放置**：手套箱/座椅下**最阴凉处**，远离太阳能板自身发热。车载 Catch-22：白天热(>45°C)充不了、夜里凉但无太阳 → 靠阴凉放置 + 大容量缓冲 + NTC 温控。
- （备选极端耐热方案 LTO/LiFePO4 因需换充电器，本方案不采用。）

## 5. 3.3V LDO（CH582M 供电）
- 选型：**XC6206P332MR**（Iq≈1µA，SOT-23-3）或 **MCP1700-330**（Iq≈1.6µA，压差更小，低压差更稳）。避免 AMS1117（Iq 5mA，吃掉休眠电流）。
- 输入接 BUS，输出 3.3V 接 CH582M VDD。
- 退耦：VDD 旁 0.1µF + 10µF 陶瓷；每电源脚就近 0.1µF。
- 注意：XC6206 压差 ~0.2V，电池 <3.5V 时 3.3V 会轻微跌落；电池近空时才发生，可接受。

## 6. 作动器（钥匙按钮模拟）
- 每路按钮：**N-MOSFET（2N7002 或 AO3400，SOT-23）** 低侧开关。
- 连接：GPIO → 100Ω 栅极串联 + 10k 下拉 → 栅极；源极接 fob 地（与 KeyGo 共地）；漏极接按钮另一触点。GPIO=高 → 导通=按下。
- 路数：**3 路**（解锁/锁车/后备箱或骑行），更多可扩展。
- 前提：KeyGo 板与钥匙板**共地焊接**（拆钥匙引线到 MOSFET）。非侵入方案需换干簧管继电器/光耦（本方案不选）。

## 7. 温度采集与 APP 上报
- **硬件安全**：CN3165 TEMP+NTC 实现 >45°C 硬停充（必做，不依赖 MCU）。
- **软件遥测**：CH582M 内置 TSENSE（ADC 内部通道），固件读取并换算温度。
  - 固件改动（后续任务）：读 TSENSE → 在 FF02 status JSON 加 `"t":<℃>`。APP `_parseSingleStatus` 已解析 JSON 字段，复用即可显示。
  - 无需外部温湿度协议；湿度无传感器（除非另加 SHT3x）。
- **可选进阶**：NTC 中点再分压一路 → CH582M ADC 引脚（高阻/PMOS 门控）→ status 加 `"tb"`（电池表面温度），比芯片温度更实用。

## 9. 布局 checklist
- RF 天线区远离 CN3165 的 SW/电源开关节点（线性无 SW，但仍远离 VIN/BAT 大电流环路）。
- CIN/CBAT 紧贴芯片引脚，地单点回流。
- CN3165 底部/周边铺铜散热（车内线性耗散最高 ~0.36W@6V/200mA）。
- NTC 物理紧贴电池外壳，走线短、远离热源。
- 电池放置区避开太阳能板直射与自身发热。
- 所有电容用 X7R，耐压 ≥16V（推荐 25V）。
